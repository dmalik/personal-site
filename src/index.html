Stack of choice

With so many tools to get the job done, a web developers choices can be overwhelming. After wading through the options, my stack of choice if I were to start a full stack web development project today would be:

Front End

React
The Componentized UI enforces workflow patterns for streamlined development.

Redux
A single store to handle all of the applications state makes it much easier to keep a mental model on mutations when developing, especially when coupled with the development tools and time travel.

PostCSS
PostCSS with its plugins and custom CSS are my choice for styling.

Back End

Horizon
Realtime back end that makes handling data easier by getting rid of writing boilerplate code.

RethinkDB
Performant, scalable, and realtime options. Required by Horizon.

Testing

Mocha
Proven, mature, JavaScript test framework with a huge ecosystem.

Expect
Simple assertion library for Mocha.

Enzyme
Helper testing utility for React.

Browsersync
Makes testing on multiple devices and browsers easy.

Other Tools

Webpack
Compiles dependencies and source into assets for testing and deployment.

Node/NPM
Package manager for using the technologies described above.

Abobe CC
Photoshop for raster graphics and Illustrator for vector graphics.

Balsamiq
Every project should start with a wireframe.
